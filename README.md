# claude-handoff-skills

`handoff.skill` is a skill file for the web/desktop versions of Claude.
- To use it, go to https://claude.ai/customize/skills (or the equivalent page in the desktop app), click "+", and select "Upload a skill."

`handoff/SKILL.md` is a skill file for Claude Code.
- To use it globally, put the `handoff` directory in `~/.claude/skills/`. As a result, the SKILL file shouuld be located at `~/.claude/skills/handoff/SKILL.md`.
- To use it per project, put the `handoff` directory in `.claude/skills` in the root of your project. As a result, the SKILL file should be located at `<project root>/.claude/skills/handoff/SKILL.md`.

Voilà!
